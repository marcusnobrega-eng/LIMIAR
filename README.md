# LIMIAR

Static public atlas for reconstructed flood signals at ANA Brazilian gauge stations.

## Website

Once GitHub Pages is enabled for the `main` branch root, the site will be served from:

`https://marcusnobrega-eng.github.io/LIMIAR/`

## What is in this repository

- the published atlas entry point: `index.html`
- frontend assets: `app.js`, `styles.css`, `vendor/`, `assets/`
- prebuilt station and status data under `data/`

This repository is the publishable website bundle, not the full Zenodo research package.

## Published data snapshot

- stations published: 2,610
- daily support range: 1911-01-27 to 2025-12-31
- month status files: 1,352
- station JSON files: 2,610
- QA panels copied: 80
